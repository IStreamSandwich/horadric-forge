# Horadric Forge Challenge

**Horadric Forge Challenge** is a small browser-based gear crafting minigame inspired by the kind of item tinkering, rerolling, upgrading, and gambling that makes ARPG crafting systems so addictive.

It is not intended to be an exact replica of Diablo 4 or any other game. It is more of a personal fan-style project built around the feeling of crafting gear, chasing better rolls, taking risks, and seeing what kind of build you end up with.

## Why I Made This

I have always enjoyed the crafting side of games like Diablo. There is something satisfying about taking a basic item and slowly turning it into something powerful through upgrades, rerolls, tempering, masterworking, and the occasional risky gamble.

I wanted to capture that feeling in a small, self-contained minigame.

The idea is simple:

> Start with basic gear, spend limited resources, make crafting decisions, take some risks, and finish with a final score and build summary.

This project is not meant to be a perfect simulation. It is more of a fun little crafting sandbox/challenge inspired by the general vibe of ARPG item systems.

## Features

- Single-file HTML/CSS/JavaScript game.
- Runs directly in a browser.
- No backend required.
- Can be hosted on GitHub Pages.
- Character name, class, difficulty, weapon style, modifier, and seed selection.
- Multiple classes including Barbarian, Sorcerer, Rogue, Necromancer, Druid, Spiritborn, Paladin, and Warlock.
- One-handed + offhand or two-handed weapon setups.
- Gear slots for a full build.
- Item Power, base stats, affixes, rarity, tempering, masterworking, capstones, and transfiguration.
- Cube recipes for adding, removing, rerolling, upgrading, and transfiguring items.
- Tuning Prisms to target affix categories.
- Class-specific affixes.
- Transfiguration outcomes with both risk and reward.
- Gear score and rank system.
- Earned badges and near-miss badge feedback.
- Local run history.
- Discord-friendly summary copy button.
- Seed sharing for challenge runs.

## How to Play

1. Start a new run.
2. Choose your class, difficulty, weapon style, modifier, and optional seed.
3. Improve each item using Cube recipes and Blacksmith actions.
4. Use materials wisely.
5. Upgrade items to Legendary.
6. Temper items, masterwork them, add capstones, and decide whether to risk Transfiguration.
7. Complete the build to see your final score, rank, badges, MVP item, and build summary.

## Difficulty Modes

### Casual

A relaxed mode with unlimited crafting resources. Good for testing, experimenting, and making silly builds.

### Standard

The main intended mode. It gives enough resources to build properly while still requiring some decisions.

### Harsh

A tighter challenge mode with fewer resources and less room for mistakes.
