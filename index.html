Choose your class, difficulty, weapon setup and run modifier. A seed is the randomiser code for the run. Reusing it lets you replay the same random sequence.

Seed tip: same seed + same class, difficulty, weapon style, modifier and action order gives the same rolls. Leave blank for a random seed.

Modifier Details
How to Play
  1. Your starter gear is equipped automatically. Improve it into the highest-scoring build you can.
  2. Use Cube recipes to add, reroll, remove and upgrade affixes.
  3. Each item has 3 temper charges. Every gear slot can use every temper category, so you can build freely without memorising slot restrictions. Casual and Standard let you choose the temper affix; Harsh rolls randomly within the chosen category.
  4. Masterwork costs Forgotten Souls and Gold. Capstone unlocks at rank 25.
  5. Transfigure last. It usually locks the item. Entropic Prism makes Transfiguration safer, but removes the bonus affix and Mythic Echo outcomes.

The Cube provides no refunds.

How to Play

Goal: build the highest-scoring gear set before your resources run out.

  • Seed: the code that drives the randomiser. Reuse it to replay a run.
  • Cube: add affixes, reroll affixes, upgrade Rares to Legendaries, or Transfigure. Transfigure can raise Item Power, empower affixes, add Capstones, or backfire. Entropic removes the worst and best extremes.
  • Blacksmith: temper each item up to 3 times and masterwork to rank 25.
  • Two-handed weapons: stronger weapon scoring, but no offhand slot.
  • Relaxed affixes: all gear slots can use all affix categories; prisms hard-target their listed categories.
  • Run modifiers: optional twists that change resources, Transfiguration risk, or scoring pressure.
  • Final totals: values are shown after Masterwork, Greater Affix, Capstone and Transfigure boosts.

Complete Build?

Final check: this will open the result screen for the current build. You can return afterwards, but the score will be saved to history.

Confirm Reforge

Warning: this will overwrite the selected gear and revert it to a Common base item. All affixes, tempering, masterworking, capstone and transfiguration effects will be removed. No materials are refunded.
